# Finestra Wealth Management

A sophisticated, architectural wealth management portal featuring immersive 3D visualizations, interactive financial instruments, and a high-fidelity institutional design.

## Features

- **Immersive Landing Experience**: A realistic ₹2000 banknote 3D animation with high-fidelity engravings and security features.
- **Architectural Service Portfolio**: Specialized interactive cards for Bonds, Securities, Mutual Funds, and Stocks.
- **Interactive Financial Visualizations**:
  - **3D Charting**: Perspective-based data flow simulations.
  - **Bond Execution**: A digital "Certificate of Indebtedness" requiring manual execution protocols.
  - **Security Vault**: A futuristic 3D visualization of zero-knowledge vault nodes.
- **Global Network Dashboard**: A professional directory of global hubs (Zürich, Singapore, New York).
- **Institutional Branding**: A custom "Gold & Obsidian" visual theme designed for the distinguished clientele.

## Tech Stack

- **React 18+** with **Vite**
- **TypeScript**
- **Tailwind CSS** (Architectural utility-first styling)
- **Framer Motion** (Sophisticated UI/UX animations)
- **Lucide React** (Consistent iconography)

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repository-url>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Design Philosophy

Finestra Wealth is built on the principle of **Architectural Precision**. The UI avoids generic patterns in favor of structured layouts that reflect the stability and clarity required in professional wealth management. Typography, spacing, and 3D depth are intentionally paired to reinforce a sense of heritage and mastery.
